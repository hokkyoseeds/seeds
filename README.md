## 更新方法

### 文章の修正
1. 該当のHTMLファイル（例：`suzuki.html`）を開く
2. 鉛筆アイコン（Edit this file）をクリック
3. 該当箇所を編集
4. ページ下部の「Commit changes」をクリック
5. 数十秒後にサイトへ反映

### 新しい研究者を追加
1. 既存ページ（例：`suzuki.html`）をベースに新規ファイル作成
2. 研究者情報を書き換え
3. 画像を `assets/` フォルダにアップロード
4. `index.html` にカードを追加

### 写真の差し替え
1. `assets/` フォルダで該当画像をクリック
2. 「Delete file」で削除 → 同じファイル名で新画像をアップロード
